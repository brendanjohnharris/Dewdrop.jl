# Stats.jl ↔ stats.py cross-validation

Validates Dewdrop's `src/Stats.jl` observables against the reference values from BrainPy/WRCircuit's
`stats.py` (a separate concern from the simulator *performance* comparison).

- `stats_reference_data.jl` — Julia literals of the reference outputs, generated by the real `stats.py`
  on a fixed input (the generator `stats_reference.py` lives in the WRCircuit repo and runs in its
  brainpy env: `.../WRCircuit.jl/.CondaPkg/.../python stats_reference.py > stats_reference_data.jl`).
- `compare_stats.jl` — runs Dewdrop's ports on the same input and asserts they match
  (`julia --project=<Dewdrop test env> compare_stats.jl`).
